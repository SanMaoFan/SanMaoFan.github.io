玩客云刷机为CasaOS系统

发表于 2024-04-06 | 分类于 硬件

场景

在清明假期闲着无聊的时候，突然想着下部 4k 电影看看，在网上找资源时，看到资源是磁力链接，而本人电脑中只有迅雷，并且没有会员，于是抱着试试的心态打开迅雷并粘贴链接，一看下载速度，我的天，竟然高达 60 kb/s，而这部电影足足有 36GB，这要下到猴年马月？我的小脑瓜子回头一想，哟吼，我还有台玩客云呢！玩客云下载时也是使用了迅雷的资源，但是用玩客云下载会比不是会员的下载更快一些（可惜不是全速），于是我掏出了在箱子里被堆压的玩客云（阉割版，战损版），接上电源，网线与硬盘，平板打开玩客云 APP，玩客云，启动！APP 页面打开了登录页面，我输入账号密码后一直提示账号密码错误，难道是记错密码了？没事，还有短信登录，于是输入手机号，点击发送验证码，可页面竟然提示短信验证码错误。。。重复多次依旧是如此，难道我的 APP 版本过低？不对，去年年末才下的版本，不可能几个月后就无法使用；带着疑问，我前往搜索引擎查找答案，才发现现在官方已经停止服务了。。。

说好的永久会员呢？对得起我们吗？卧槽！

这么说，这个机器当前已经算是废品了，用都用不了，可惜咯；可这怎么会难道一个程序员呢？玩客云原系统不能用，但是换个系统，那又是好好的 NAS 啦，接着在互联网上转了一圈后，发现有一个系统能轻松刷机，那就是 CasaOS 系统，而要刷入 CasaOS 系统前，需要先将玩客云刷入环境。

刷入 armbian 系统

1、刷机需要一个系统：armbian，系统可前往 armbian 的 github 地址 下载，我选用的是 Armbian-unofficial_24.5.0-trunk_Onecloud_bullseye_current_6.6.21_minimal.burn.img.xz 的资源，下载后解压出来备用。

2、接着下载一个烧录工具，这里下载的是 Amlogic_USB_Burning_Tool 工具，版本为 v2.1.6.8，将其安装在电脑上后打开，选择文件》导入烧录包，这里的烧录包就是步骤1中下载好并解压后的系统。

3、接着将玩客云拆卸并取出 PCB 板，找个金属镊子或者是导电的工具，因为刷机需要连接玩客云 PCB 上两个短接触点的位置，这样才能正常刷机，而玩客云 1.0 和其他版本的机器的 PCB 板上的短接触点的位置不相同，以下为 1.0 版本短接触点的位置：

而另一个版本的短接触点位置如下所示：

4、使用一根公对公的 USB 线，一头插入电脑，一头插入 PCB 板的靠近 HDMI 口的 USB 口，如果电脑有 typec 口，也是可以使用 typec 对 USB 口来连接的。

4、准备好后，先将电脑上的烧录工具将其选项设置好，并点击开始按钮

接着，将 PCB 板上的短接触点连接，最后启动玩客云的电源，然后看电脑上的烧录工具是否有进度条变动即可，例如：

只要进度条有启动，这个时候就可以松开短接触点的连接了，不然可能造成烧录工具报错并重新开始烧录。需要注意的是，在所有前置步骤准备好后才启动电源，不然可能失败，而所有工作准备完毕了，可是启动电源后，烧录工具没有任何变化，可能是短接触点接触不良，可以的话，可将其表面焊锡或是覆盖物刮掉一点后再进行连接；烧录过程中一直失败的话，不用担心，关闭电源，取消 USB 连接，再次重复以上步骤即可。最后便可以等待系统的烧录完成。如果一直烧录不成功，可能内置的 emmc 有损坏，这个我无能为力。

5、系统烧录完成后，需要先点击烧录工具的停止按钮，再关闭烧录软件。

安装 CasaOS 系统

在刷入 armbian 系统后，将 PCB 连接上网线，然后在电脑的浏览器进入 WIFI 的管理控制台，每个 WIFI 机器进入控制台的 IP 具体以个人为准，在进入 WIFI 控制台后，可以找到分配给玩客云的 IP，玩客云的主机名称一般显示为 onecloud，记录下这个主机对应的 IP，接着使用远程登录工具登录该 IP 即可，默认用户名为root，默认密码为1234，端口默认22；首次进入系统，会提示修改密码，输入好新密码后回车，会提示是否要创建新的用户，当前使用 root 账户即可，所以 ctrl + c 退出创建新用户步骤。

基本设置完毕后，需要输入 date -R 命令来查看当前时区时间是否正确，如果时区不对，则使用 cp /usr/share/zoneinfo/Asia/Shanghai /etc/localtime 命令来修改时区时间，注意，cp 后是一个空格，而 /etc/localtime 的前面是两个空格，输入错命令会提示这个：

设置后如果没有错误，则再次输入 date -R 查看设置的时间时区是否正确。

接着是添加系统的软件源，这可以提高安装的成功率，输入 nano /etc/apt/sources.list，进入编辑后，将所有代码的开头加上 # 符号，也就是注释该代码，已有 # 的代码不动，接着把光标移动到最下面的空白处，将以下代码粘贴进去：

可以将以上代码中带了 # 符号的代码去除 # 符号，这样会有更多的备用软件源供使用，设置好后，按下 ctrl + x 来退出，这时编辑软件会提示是否保存内容，按下 y 键即可保存。

接着是安装 CasaOS 系统，只需将以下代码输入终端并按下回车即可进入等待: wget -qO- https://get.casaos.io | bash 。

等待的时间在十分钟左右，在安装完毕后，系统会给出登录的 IP 地址：

最后在浏览器中输入登录的 IP 地址，并创建新用户及其密码即可进入系统内部。

到此为止，一切就完成了，废品玩客云便又成了易用的 NAS 服务。
